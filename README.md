# mine-t01
This repository provides a centralized, version‑controlled space for storing, organizing, and sharing datasets, scripts, and reference materials used in the development of groundwater flow and contaminant transport models. The project supports an ongoing PhD research effort focused on understanding the hydrologic behavior of mining residues, the migration of associated contaminants, and the potential risks these pollutants pose to surrounding groundwater systems.

The repository is intended to:

Maintain clean, traceable datasets for model construction, calibration, and scenario analysis.

Host numerical modeling inputs and outputs (e.g., MODFLOW, MT3DMS/MT3D‑USGS, SEAWAT, or other tools).

Provide documentation, figures, and workflow notes that support reproducibility and transparency.

Facilitate collaboration between the student, advisors, and technical contributors throughout the research and dissertation process.

By consolidating all modeling resources in one place, this repo helps ensure consistency, reproducibility, and efficient progress toward the dissertation defense.
